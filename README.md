# Araahi вЂ“ Accommodation Listing Platform

Araahi is a full-stack property listing platform inspired by Airbnb. Built with Node.js, Express.js, MongoDB, Mongoose, EJS, and Joi, it allows users to create, view, update, and delete accommodation listings. The application features server-side validation, custom error handling, and RESTful architecture for efficient property management.

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Mahi02042006/Araahi-Accommodation-Listing-Platform.git
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Ensure MongoDB is Running Locally

Start your local MongoDB server before running the application.

### 4. Start the Application

```bash
node app.js
```

### 5. Open in Browser

```text
http://localhost:8080/listings
```
